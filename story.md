## Recent Features

### F1 Lap Improvement Analysis API
Develop an API that processes Formula 1 pole position lap data, specifically throttle and braking timeseries. The API will analyze this data to identify and return specific time windows within the lap where improvements could be made, based on the provided timeseries data.
